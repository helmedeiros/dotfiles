# tapeit

The [`tapeit`](https://github.com/helmedeiros/tapeit) CLI.

## What this topic does

- **`install.sh`** (run by `bin/dot`) `go install`s the `tapeit` binary into
  `$GOPATH/bin`, which `go/path.zsh` already puts on `PATH`.

The tool's source lives in `~/Dropbox/PROJECTS/tapeit` for development.
